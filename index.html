    Steps!

    Step 1: Accessing the file

  • click on the link labeled "TEMPLATE"
  • Fork the repl, creating your own copy
  • navigate your way to the HTML file, this is the file you're gonna use mainly

    • Step 2: text

  • In the HTML file navigate to the body
  • in the following tags: h1, h2, h3, p replace the sample text with your own!
  • If you want to add more text on a new line, you can add any of the 4 tags above and place your text within!
  • to change font size/color, find the respective tags in the css and add "font-size: ??px; || color: blue;"

    • Step 3: images

  • Next to the text "Files" at the top of the page, look for the 3 dots and upload your image file there
  • Next, navigate to the HTML body, then look for the "img" tags
  • Inside image tag, look for the "src=" line
  • In the following quotes, replace the image with your image file (ex.png)
  • To change the main background image on the main page, navigate to the style.css file and search for the line: "background-image: url();"
  • In here, replace the "" with your image file or url!
  • *Make sure not to forget to edit the background in the media query aswell!*

    • Step 4: boxes

  • In the HTML body, find the container with the id="boxes"
  • Inside this container, there are multiple containers with the class="box"
  • From here, you can copy paste the entire div class="box" container and paste more within the section
  • With the new boxes, you can edit the text and images within!

    • Step 5: links

  • Navigate to the Newsletters html file
  • look for the "a" tags
  • here, you can replace the text within the "" after href and insert your own link!

    • Step 6: general styling

  • Navigate to the style.css file
  • here, theres a list of all the tags used in all the html files
  • within the {}, you can edit the text to change the appearance of the objects!
  • You can search styles up on google and paste them here!
